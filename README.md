# Handshake Autofill

*"Commit your work to the Lord, and your plans will be established." - Proverbs 16:3*

Handshake Autofill is a Chrome Extension designed to streamline the evaluation and task submission process on Handshake AI. By automating repetitive form-filling such as calculating quality scores, rating counts, and pre-filling task labels, this extension saves significant time and reduces manual entry errors.

## Features
- **Profile Presets**: Save up to three distinct profiles to quickly switch between different task types and rating configurations.
- **Global Settings**: Quality scores and good/fair/bad counts remain consistent across profile swaps to maintain steady grading criteria.
- **Automated Form Advance**: Detects multi-step screens (like the "Continue" wall) and naturally advances the Handshake AI form to completion.

## Installation

Due to Chrome's strict security policies, custom extensions not hosted on the Chrome Web Store must be loaded in through "Developer Mode" manually. 

### Manual Installation (Mac / Linux / Windows)
1. Download or clone this repository to your computer.
2. Open Google Chrome and navigate to `chrome://extensions/` in your URL bar.
3. In the top right corner, toggle **Developer mode** to **ON**.
4. Click the **Load unpacked** button that appears in the top left.
5. Select the `autofill-handshake` folder that you downloaded.
6. The extension is now installed! You can pin the Handshake icon to your toolbar for easy access.

## Usage
1. Click the Handshake Autofill icon in your Chrome toolbar.
2. Select a **Profile Preset** to save your current task settings to.
3. Configure your Task Type, Quality Score, and Ratings.
4. Click **Save Defaults** to save the Task Type to your chosen profile.
5. When on an active Handshake task page, click **Autofill Page** to automatically fill the form and advance.
